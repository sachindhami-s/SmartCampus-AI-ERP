# 🎓 SmartCampus AI ERP

An AI-powered Full Stack College ERP System built using the MERN Stack.

## Features

- Student Portal
- Faculty Portal
- Admin Dashboard
- Attendance Management
- Assignment Management
- Notes Management
- Result Management
- Timetable Management
- AI Study Planner
- AI Chatbot
- AI PDF Summarizer
- AI Quiz Generator
- AI Doubt Solver

## Tech Stack

- React.js
- Node.js
- Express.js
- MongoDB
- Tailwind CSS
- JWT Authentication
- Gemini AI API
